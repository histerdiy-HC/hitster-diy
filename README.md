# HITSTER-DIY
Generador de cartas imprimibles para tu propio juego de mesa tipo Hitster, basado en tus playlists favoritas de Spotify. Este script toma una lista de reproducción, obtiene los datos de las canciones y genera un PDF profesional listo para imprimir, cortar y jugar. ¡No necesitas comprar el juego original!
## Caracteristicas
·Print-Ready: Genera PDF A4 optimizado para impresión dúplex (frente y dorso).

·Diseño Personalizado: Cartas con diseño estilizado y año de solución.

·Fácil de usar: Scripts en Python para automatizar el proceso.
